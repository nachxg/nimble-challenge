# Challenge Nimble Gravity

Este programa se realizó para enviar mi postulación a la posición Full Stack Developer en Nimble Gravity.

Consiste en una aplicación web desarrollada en React que consume una API REST para obtener los datos del candidato y el listado de posiciones disponibles. Cada posición se muestra en una tarjeta con un campo para ingresar la URL de un repositorio de GitHub y un botón para enviar la postulación.
